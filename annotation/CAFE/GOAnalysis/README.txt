1. getlist.php converts CAFE output into gene lists for each node.
2. Output for each node: [NodeName].expandedgenes.famsizefilter[FamSizeFilter].txt and [NodeName].contractedgenes.famsizefilter[FamSizeFilter].txt. These lists can be used for GO analysis.
3. countsigexpcontractions.php reports counts of significantly expanded/contracted gene families from the cafe output.
